## Примеры диалогов

### Пример 1: Вопросы про программирование

```
👤 Вы: Объясни что такое корутины в Kotlin

🤖 YandexGPT: Корутины (Coroutines) в Kotlin - это легковесные потоки, 
которые позволяют писать асинхронный код в последовательном стиле. 
Основные преимущества:
- Легковесность (можно создать тысячи корутин)
- Структурированная конкурентность
- Простой синтаксис с suspend функциями
- Встроенная поддержка отмены и обработки ошибок
```

## Особенности

✅ **Что работает:**
- Отправка и получение сообщений
- История чата в рамках сессии
- Автоматическая прокрутка к новым сообщениям
- Индикатор загрузки во время ответа AI
- Красивый Material Design 3 интерфейс
- Работа без VPN из России

⚠️ **Ограничения:**
- История не сохраняется между запусками
- API ключ нужно вводить каждый раз
- Максимум 2000 токенов на ответ
- Нет поддержки изображений

## Производительность

- **Время ответа**: 2-10 секунд (зависит от сложности вопроса)
- **Размер APK**: ~25 МБ
- **Минимальная версия Android**: API 34 (Android 14)
- **Использование памяти**: ~50-100 МБ
- **Сетевой трафик**: ~1-5 КБ на запрос/ответ

## Безопасность

🔒 **Как хранятся данные:**
- API ключ хранится только в памяти приложения
- При закрытии приложения всё стирается
- Нет записи на диск
- Нет отправки данных третьим сторонам

⚠️ **Рекомендации:**
- Не делитесь API ключом
- Используйте сервисные аккаунты с ограниченными правами
- Следите за использованием квоты
- Для продакшена используйте серверную часть
